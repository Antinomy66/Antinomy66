# 我的第一篇博客

你好！这是我的个人博客的第一篇文章。

## 关于这个网站

这个网站是我的个人空间，用来记录我在学习和工作中的心得与收获。

### 主要内容

- **算法学习**：刷题记录与算法总结
- **Hadoop 笔记**：大数据学习笔记
- **个人博客**：日常心得和分享

## 为什么写博客

写博客不仅可以巩固学过的知识，还能帮助他人。正如费曼学习法所说：

> 如果你不能简单地解释它，你就没有真正理解它。

## 技术栈

本站使用纯前端技术构建：
- HTML + CSS：页面结构和样式
- JavaScript：交互逻辑
- Markdown：笔记编写
- marked.js：Markdown 渲染
