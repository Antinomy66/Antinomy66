# 二分查找详解

二分查找（Binary Search）是一种在有序数组中查找目标值的高效算法。

## 时间复杂度

- 最坏情况：O(log n)
- 最优情况：O(1)

## 基本模板

`python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = left + (right - left) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
`

## 注意事项

1. **边界条件**：while left <= right 还是 left < right，要根据具体情况选择
2. **中间值计算**：mid = left + (right - left) // 2 可以防止整数溢出
3. **有序前提**：二分查找的前提是数组已经有序

## 变种问题

| 类型 | 说明 |
|------|------|
| 查找第一个等于目标的位置 | right = mid 收缩右边界 |
| 查找最后一个等于目标的位置 | left = mid 收缩左边界 |
| 查找第一个大于等于目标的位置 | 标准 lower_bound |
| 查找最后一个小于等于目标的位置 | 标准 upper_bound |

二分查找虽然思路简单，但细节处理非常容易出错，建议熟记模板。
